## 🎨 Design & UX
O site foi construído seguindo princípios de design de alto nível:
- **Tipografia:** Uso da fonte *Instrument Serif* para títulos e *Inter* para leitura técnica.
- **Visual:** Estética Dark Mode com acentos em metal líquido (Liquid Chrome).
- **Interatividade:** Animações suaves de revelação ao rolar (Scroll Reveal) e botões com efeitos de brilho intenso (Glow).
- **Responsividade:** Experiência otimizada para Desktop, Tablet e Mobile.

## 🛠️ Tecnologias Utilizadas
- **Linguagens:** HTML5, CSS3, JavaScript (Vanilla).
- **Formulário:** Web3Forms (Processamento de e-mails via Fetch API).
- **Mockups:** Shots.so (Apresentação de projetos em dispositivos reais).
- **Hospedagem:** GitHub Pages.

## 📁 Estrutura do Projeto
- `index.html`: Estrutura semântica do portfólio.
- `style.css`: Estilização completa, variáveis CSS e animações.
- `script.js`: Lógica de scroll suave, validação de formulário e animações de entrada.
- `logo.png`: Identidade visual em alta resolução com fundo transparente.
